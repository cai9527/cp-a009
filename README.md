## 产品 PRD 设计

### 核心功能点
- 用户注册与登录系统：支持手机号注册、登录、个人资料管理
- 运动数据采集：支持跑步、骑行、步行等多种运动类型
- 实时追踪：运动过程中实时记录距离、时长、配速、卡路里等数据
- 运动历史记录：查询历史运动记录，支持按时间、类型筛选
- 数据可视化：图表展示运动趋势、周/月/年度统计
- 运动目标设定：设定每日/每周运动目标，跟踪完成情况
- 个人中心：用户信息管理、设置、数据导出

### 页面列表
| 页面名称 | 目录名 | 页面类型 | 功能点列表 |
|---------|--------|----------|----------|
| 首页 | home | tabBar页面 | 今日运动概览、快速开始运动入口、今日数据统计卡片、最近运动记录 |
| 运动 | sport | tabBar页面 | 运动类型选择（跑步/骑行/步行）、实时运动追踪、运动数据面板、暂停/结束控制 |
| 统计 | stats | tabBar页面 | 周/月/年数据统计、趋势图表、运动类型分布、总里程/时长/卡路里统计 |
| 我的 | mine | tabBar页面 | 用户信息展示、目标设定、运动成就、设置入口、登录/注册入口 |
| 登录页 | login | 二级页面 | 手机号登录、注册入口、忘记密码 |
| 注册页 | register | 二级页面 | 手机号注册、验证码、设置密码 |
| 运动详情页 | sport-detail | 二级页面 | 单条运动记录详情、运动轨迹、详细数据 |
| 目标设定页 | goal-setting | 二级页面 | 设定每日/每周运动目标、目标类型选择 |
